## 1.2 目录

1. bin

   放置脚本文件，如启动脚本elatsearhc，插件安装脚本

2. config

   elasticsearch .yml，elasticsearch 配置文件，如集群配置、JVM配置

3. jdk

   java运行环境

4. data

   path.data，数据持久化文件

5. lib

   依赖的相关类库

6. logs

   path.logs，日志文件

7. modules

   包含的所有ES模块

8. plugins

   包含的所有的已安装的插件
